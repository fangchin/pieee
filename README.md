# Proceedings of The IEEE (PIEEE) papers

This directory is for materials related to papers written by Chin and
his co-authors. For now, it contains just two subdirectories, Files
and Images. In the future, each paper will get a separate
directory. Each directory contains its own Images and Files.

You are advised to clone the entire repo and then on a Linux or UNIX
(including Apple OS X), use CLI tools to process further.
